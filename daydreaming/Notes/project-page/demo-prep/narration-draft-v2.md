# Narration Draft V2

Matches the v3 demo page: exploded cycle + arc cards + causal strips.
~75 seconds total. Tell Tony's story, land one technical point.

---

## Title card (5 sec)

"This is Latent Dreamer. I'm building a cognitive engine — a
persistent inner life for characters — based on Erik Mueller's 1990
daydreaming architecture, reconstructed in Clojure."

## Exploded cycle 6 (20 sec)

"Here's what one cycle looks like. Tony is a 7-year-old from a Mark
Friedberg script. He's neurodivergent — his sensory processing
overwhelms him. In this cycle, four different kinds of thinking are
competing for his attention. Rehearsal wins — he's practicing Monk's
rhythm, finding a count. And when we check how the mural reads after
that... same mural, same cops, same lights — but it flips from threat
to challenge. Not because the world changed. Because Tony did."

## Arc cards, scrolling (25 sec)

"That's one cycle. Over twenty, it becomes an arc. He starts
overloaded. Rationalization tries to reframe things. Reversal replays
what went wrong. Rehearsal builds control. The street knocks him back.
But each recovery is stronger. By the end, the same street that
overwhelmed him can't undo what the apartment built."

## End card (10 sec)

"None of this was scripted. The kernel selected what to think about.
The families competed. The state accumulated. The prose was generated
from that state. Next step is connecting this to Scope for real-time
rendering."

## What makes this interesting (reminder to self)

The technical content boils down to one sentence:

"Four kinds of thinking compete, the winner changes Tony's state,
and that changes how the same place reads next time."

Everything else is Tony's story.

The thing that makes it interesting isn't the machinery — it's that
the arc is real. It wasn't authored. A cognitive engine produced a
developmental trajectory that reads like a character growing. That IS
the demo, even without video rendering hooked up.

"The mind works. The stage is next."

---

## The key paragraph if someone asks "why does this matter?"

A language model can generate beautiful inner-life prose from a
character sheet. But it has no concerns between calls. No emotional
state that decays. No memory that accumulates and earns trust. No
sense that the same place should read differently because of what
happened three cycles ago. The kernel provides those things. The LLM
inhabits the state it's given. Together they produce path-dependent
inner life — not a better prompt, but an actual developmental arc.
